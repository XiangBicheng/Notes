## 3. 显示转换

### 3.1 命名的强制类型转换

- 一个命名的强制类型转换具有如下形式：*cast-name\<type\>(expression)*。其中，type是转换的目标类型，expression是要转换的值，如果type是引用类型，则结果是左值。cast-name是**static_cast**、**dynamic_cast**、**const_cast**和**reinterpret_cast**中的一种。

### 3.2 旧式的强制类型转换

- 在早期的C++语言版本中，显式地进行强制类型转换包含两种形式：

  ```c++
  type (expr)； // 函数风格
  (type) expr;  //C语言风格
  ```
